# 此项目为小程序商城后台管理系统
1. 该项目使用的技术栈为Vue全家桶 + ES6 + Element UI + Axios
2. 包含了商品分类，商品管理，商家管理，订单管理和用户管理等子模块
3. 采用模块化、组件化、工程化的模式开发（模块化、组件化和工程化详解戳：[谈谈前端工程化、组件化、模块化以及自动化](https://www.xcio.cn/index.php/archives/62/).）

## 项目依赖安装
```
npm install

cnpm install

yarn install
```

### 启动项目（开发环境）
```
npm run serve
```

### 项目打包（生产环境打包）
```
npm run build
```

### 生产环境运行
```
serve -s dist
```

### Lints and fixes files
```
yarn lint

```
